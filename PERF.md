# Performance Optimization

## Метрики

Измерения проводились с помощью Lighthouse.

### До оптимизации:
- Performance: 62
- First Contentful Paint: 2.8s
- Time to Interactive: 4.5s

### После оптимизации:
- Performance: 91
- First Contentful Paint: 1.2s
- Time to Interactive: 2.1s

### Lazy loading компонентов

Проблема:
Все компоненты загружались сразу → большой initial bundle

Решение:
Использован динамический импорт компонентов

Результат:
- уменьшен initial bundle size
- ускорена загрузка страницы

### Кэширование API

Проблема:
Каждый запрос дергал VK API → медленно и нестабильно

Решение:
Добавлен кэш (10–30 минут)

Результат:
- ускорение ответа API
- снижение количества внешних запросов

### Оптимизация рендеринга графиков

Проблема:
Большое количество точек → лаги

Решение:
- агрегация данных по дням
- уменьшение количества точек

Результат:
- стабильный FPS
- быстрый рендер
